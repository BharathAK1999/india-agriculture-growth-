# india-agriculture-growth-
# 🌾 India Agricultural Production Forecasting (DSML Project)

## 📌 Overview
This project analyzes historical agricultural production in India and predicts future trends (2024–2030) using Machine Learning and Time Series models.

## 📊 Dataset
- Source: FAOSTAT
- Time Range: 1961–2023
- Country: India

## 🔍 Key Analysis
- Production growth trends
- Top crop analysis
- Year-wise performance

## 🤖 Models Used
- Linear Regression
- Random Forest
- Support Vector Machine (SVM)
- SARIMA
- Prophet
- LSTM (Deep Learning)

## 📈 Forecast
- Predicted production from 2024 to 2030
- Visualization of future trends

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn
- Statsmodels
- Prophet
- TensorFlow

## 🚀 Results
- Strong upward trend in agricultural production
- Forecast indicates continued growth with possible environmental constraints
